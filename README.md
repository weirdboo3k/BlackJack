# Black Jack Game / ブラックジャックゲーム
A simple Black Jack card game developed using JavaScript.  
Players can play against the dealer following standard Black Jack rules.
JavaScriptで開発したシンプルなブラックジャックゲームです。  
基本的なルールに基づき、ディーラーと対戦できます。
## Features / 機能
### Start a game / ゲーム開始
Players can start a new game and receive initial cards.  
ゲームを開始すると、プレイヤーにカードが配られます。
### Hit / Stand / ヒット・スタンド
Players can choose to draw another card (Hit) or stop (Stand).  
ヒットでカードを追加、スタンドで行動を終了します。
### Dealer logic / ディーラー処理
The dealer automatically draws cards based on Black Jack rules.  
ブラックジャックのルールに基づき、ディーラーが自動でカードを引きます。
### Win / Lose judgment / 勝敗判定
The game calculates scores and determines win, lose, or draw.  
スコアを計算し、勝敗を判定します。
